# Dynamic Functional Connectivity for Identifying Resting-State Biomarkers of Autism Spectrum Disorder
By Arryn O'Brien

## How to Use (Windows)
1. Open a terminal
2. Navigate to the git repository directory
3. Run the command `python download_abide_preproc.py -d func_preproc -p ccs -s filt_global -o func_preproc_css_filt_global` to download the data from ABIDE Preprocessed (will take a while)
4. Run the code in main.ipynb
